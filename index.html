Swagger Maven Plugin

This plugin can let your Swagger annotated project generate Swagger JSON and your customized API documents in build phase.

You must already known what is Swagger JSON, check the results of this plugin generates here to see if it is what you want.

Usage

Import the plugin in your project by adding following configuration in your plugins block:

<build>
    <plugins>
        <plugin>
            <groupId>com.github.kongchen</groupId>
            <artifactId>swagger-maven-plugin</artifactId>
            <version>${swagger-maven-plugin-version}</version>
            <configuration>
                ...
            </configuration>
        </plugin>
    </plugins>
</build>

Versions

  • 3.0-SNAPSHOT supports Swagger Spec 2.0 (DRAFT)
  • 2.3.2 supports Swagger Spec 1.2 (ACTIVE)
  • 1.1.1 supports Swagger Spec 1.1. (No longer maintained)
